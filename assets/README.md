# Assets Guide

This folder is ready for optional profile visuals that can replace or extend the current README setup.

Suggested files:

- `banner.png`
  Recommended use: replace the external capsule banner at the top of the profile.
  Recommended size: around `1500x500` or another wide banner ratio.

- `profile-avatar.png`
  Recommended use: static local profile visual near the introduction.
  Recommended size: around `512x512`.

- `avatar.gif`
  Recommended use: optional animated alternative to `profile-avatar.png`.

- `tech-loop.gif`
  Recommended use: optional decorative animation below the tech stack section.

The current README already renders correctly without these files because it uses external GitHub-safe services. Add the local assets only when you are ready to replace or extend the visuals.
