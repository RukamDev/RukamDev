# Icons Guide

This folder is ready for optional animated or custom tech icons.

Suggested filenames:

- `java.gif` or `java.svg`
- `mysql.gif` or `mysql.svg`
- `python.gif` or `python.svg`
- `php.gif` or `php.svg`
- `javascript.gif` or `javascript.svg`
- `typescript.gif` or `typescript.svg`
- `c.gif` or `c.svg`
- `cpp.gif` or `cpp.svg`
- `git.gif` or `git.svg`
- `github.gif` or `github.svg`
- `vscode.gif` or `vscode.svg`
- `intellij.gif` or `intellij.svg`

Recommended approach:

1. Keep the current `skillicons.dev` fallback in the README until your local assets are ready.
2. Replace each skill row in `README.md` with local `<img src="./assets/icons/...">` tags when you have polished SVG or GIF files.
3. Prefer SVG for crisp rendering and GIF only when motion is essential.
